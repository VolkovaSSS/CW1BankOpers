# **Приложение для анализа банковских операций из файлов EXCEL**
## Описание
- получает данные из файлов формата excel
- анализирует транзакций
- генерировать JSON-данные для веб-страниц
- формировать Excel-отчеты
- другие сервисы
## Структура проекта
- main.py - реализована основная логика проекта
- src/views.py - генерировать JSON-данные для веб-страниц (сводные данные по картам, топ транзакций, актуальные курсы валют и стоимость акций по настройкам пользователя) 
- src/services.py - Инвесткопилка. Позволяет копить через округление ваших трат
- src/reports.py - Траты по выбранной категории на заданную дату
- src/utils.py - вспомогательные функции
- data - файлы с данными о транзакциях и настройками пользователя
- tests - папка с тестами

## Установка
- Клонируйте репозиторий:
```git clone git@github.com:VolkovaSSS/pythonProject1.git```
- Установите зависимости:
```pip install -r requirements.txt```
- создайте файл .env по шаблону .env.example
## Использование
Для использования нужно запустить файл main.py
например: python main.py
### Зависимости
- Python 3.x
- requests = ">=2.25.1"
- flask = ">=2.0.1"
## Тестирование
При тестировании использовался фреймворк pytest
Тестами охвачены модули:
- views
- services
- processing
- reports
- utils


Для запуска тестов выполните команду:
```sh
pytest --cov=src --cov-report=html
```
